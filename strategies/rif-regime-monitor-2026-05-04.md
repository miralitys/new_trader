## Interpretation

- RIF **не подходит** как always-on: 730d = -41.82%, MaxDD 73.90%.
- RIF **становится рабочим кандидатом** только как regime-monitor: health30_60 дает 730d +48.19%, MaxDD 15.48%, PF 1.37.
- Самый аккуратный режим: health30_60 + weekly kill 2%, 730d +39.27%, MaxDD 8.29%, PF 1.67.
- Health-gate считается только по прошлым данным, без подглядывания в будущее.

## Working Candidate

Текущий лучший рабочий вариант:

- execution: strict maker, fee 0.02%, limit offset 0.05%

- health30: return > 0, PF >= 1.10, MaxDD <= 15%, trades >= 20
