# Минутка Cashflow 3 - ONE / RIF / SPELL Defensive 10%

Статус: `главная cashflow-стратегия проекта / защитный 10% кандидат`

Тип: портфельная минутная cashflow-стратегия.

Это более спокойная версия 10%-cashflow идеи. Она не пытается выжать максимум прибыли, а снижает просадку относительно первой версии ONE/RIF/SPELL.

## Состав

- `ONE` - вес `12%`;
- `RIF` - вес `12%`;
- `SPELL SHORT Best` - вес `75%`.

Сумма весов округлена из тестового CSV: `ONE 12% / RIF 12% / SPELL 75%`.

## Фиксированные параметры

| Параметр | Значение |
|---|---:|
| Timeframe | `1m` |
| Portfolio | `ONE 12% / RIF 12% / SPELL 75%` |
| Scale | `5.5x` |
| Monthly target | `$100` от `$1000`, то есть `10%` |
| Monthly loss stop | `30-35%` |
| Base fee | maker `0.02%` за сторону |
| Base slippage | `0` |
| Entry | strict maker-limit / post-only pool |
| Обязательное правило | после достижения месячной цели торговлю остановить до следующего месяца |

## Вывод

Эту стратегию можно считать одной из основных, но именно как защитную base-maker версию.

Она лучше v2.1 по просадке, но хуже по суммарной прибыли. Поэтому правильное место этой стратегии в проекте:

- не high-risk;
- не stress-proof;
- не замена всем cashflow-стратегиям;
- да, основная защитная 10%-cashflow версия при аккуратном maker/post-only исполнении.

Перед реальными деньгами обязательны:

- ежедневный paper/post-only контроль;
- проверка фактического fill-rate;
- остановка после достижения месячной цели;
- отдельный контроль комиссии и funding, если торгуется futures.
