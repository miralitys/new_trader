# AMP Strategy Search

Дата проверки: 2026-05-03

Цель: проверить `AMP` на той же минутной механике, что и остальные монеты.

## Итог

`AMP` сейчас нельзя честно протестировать в текущей futures-методологии.

Причина: для `AMPUSDT` не найдены свежие Binance Futures archive-файлы.
Скрипт поиска остановился на загрузке данных:

```text
RuntimeError: No recent futures archive files found for AMPUSDT
```

## Что проверено

- Символ: `AMPUSDT`
- Таймфрейм: `1m`
- Требуемый рынок: Binance Futures archive
- Окна: `7/30/60/90/180/365`
- Результат: данные не загрузились

Старый мульти-прогон GALA-шаблонов тоже дал ошибку по всем окнам:

| Стратегия | 7d | 30d | 60d | 90d | 180d | 365d |
|---|---:|---:|---:|---:|---:|---:|
| 7.3 | error | error | error | error | error | error |
| 10 | error | error | error | error | error | error |
| 11.2 | error | error | error | error | error | error |

## Дополнительная проверка доступности

Из текущего окружения Binance global API для futures/spot вернул HTTP `451`.
На Binance.US символ `AMPUSDT` не найден.

Это значит, что сейчас нет надежного источника минутных данных `AMPUSDT`,
который был бы сопоставим с тем, как мы проверяли GALA, CHZ, SHIB, JASMY,
SAND, MANA, ANKR и другие futures-монеты.

## Вывод

`AMP` не добавляем в рабочие монеты.

Это не означает, что стратегия на AMP плохая. Это означает другое:
на текущих данных ее нельзя честно проверить. Для нашего списка рабочих
монет статус такой:

`AMP - нет futures-данных / не брать в текущей системе`.
