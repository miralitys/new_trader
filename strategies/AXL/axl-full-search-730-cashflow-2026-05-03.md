# AXL — полный поиск, sweep позиции, 730 дней и cashflow

Дата проверки: 2026-05-03.

Статус: **отдельная SHORT watchlist-стратегия, не брать в работу сейчас**.

Короткая метка: `AXL отдельная стратегия / WATCHLIST / режимная, не рабочая`.

Причина: на `365d` результат красивый, но на `730d` стратегия уходит в минус и показывает слишком большую просадку.

## Что проверяли

Монета: `AXLUSDT`.

Данные для полного поиска:

- Свечей: `1,061,280`
- Период: `2024-04-26 00:00 UTC` - `2026-05-02 23:59 UTC`
- Таймфрейм: `1m`
- Исполнение: maker-limit у текущей цены
- Комиссия базовая: `0.02%` за сторону
- Проскальзывание базовое: `0`
- Daily kill stop: `2%`

Полная сетка:

- LONG/SHORT
- thresholds: SHORT `40/50/60`, LONG `50/60/70`
- режимы: `base/wide`
- TP: `0.35% / 0.50% / 0.70% / 1.00% / 1.20%`
- SL: `4.00%`
- time stop: `60/90/120/180 минут`

## Как отдельная стратегия

Если рассматривать AXL отдельно, без GALA/SPELL, то текущий статус такой:

| Пункт | Значение |
|---|---|
| Название | AXL SHORT Watchlist |
| Логика | SHORT `threshold 40`, режим `wide` |
| TP / SL | `1.00%` / `4.00%` |
| Time stop | `180 минут` |
| 365d | `+110.31%`, MaxDD `25.60%`, PF `1.12` |
| 730d | `-54.97%`, MaxDD `84.43%`, PF `0.92` |
| Решение | **Не рабочая сейчас. Только watchlist / режимная идея** |

Отдельно торговать AXL по этой версии нельзя: последний год был сильный, но двухлетний тест показывает, что edge нестабильный.

## Итог

AXL пока **не подходит**:

- На 365d AXL дает красивый результат.
- На 730d стратегия уходит в сильный минус.
- Даже позиция 25% остается отрицательной на 730d.
- Stress комиссии/исполнения почти полностью съедает edge.
- В cashflow AXL ухудшает GALA/SPELL: было `24/24` месяцев, становится максимум `19/24`.

Решение: **AXL не добавлять к GALA/SPELL cashflow и не брать как отдельную рабочую стратегию**. Оставить как rejected/watchlist-кандидат, но не как рабочую стратегию.
