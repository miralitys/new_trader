# Binance New Coin Candidates

Дата отбора: 2026-05-03.

Статус: кандидаты для проверки, не зафиксированные стратегии.

## Источники

- Binance Data Vision futures archive:
  `https://data.binance.vision/?prefix=data/futures/um/monthly/klines/`
- Binance S3 public futures archive listing:
  `https://s3-ap-northeast-1.amazonaws.com/data.binance.vision?delimiter=/&prefix=data/futures/um/monthly/klines/`
- Binance spot data API used as liquidity proxy:
  `https://data-api.binance.vision/api/v3/ticker/24hr`

Binance Futures REST API из текущего окружения вернул `451`, поэтому для
отбора использовался официальный public-data архив Binance. Это нормально для
нашей задачи: нам важнее не только наличие символа, но и наличие исторических
1m-свечей.

## Критерии отбора

Монета должна:

- не входить в уже проверенный список;
- иметь Binance Futures 1m archive минимум с 2024-05 по 2026-04;
- быть USDT perpetual-символом в архиве;
- иметь текущую spot-ликвидность как ориентир;
- быть достаточно волатильной для минутной стратегии;
- не быть слишком тяжелой монетой типа BTC/ETH/SOL, потому что наша механика
  лучше работала на более спекулятивных альтах.

## Что делать дальше

Первая честная проверка:

1. Прогнать первую очередь на 7/30/60/90/180/365 дней.
2. Оставить только те, где годовой PF выше 1.10-1.15 и стресс комиссии не
   убивает результат.
3. Затем добавить лучшие из них в monthly cashflow search.
4. Главный критерий не доходность, а месяцы `24/24`, MaxDD, PF и устойчивость
   после ухудшения комиссии/slippage.

Мой приоритет для следующего запуска:

`1000LUNC`, `XVG`, `REZ`, `BB`, `AI`, `ZEN`, `KNC`, `ORDI`, `AXL`, `RLC`,
`BAT`, `API3`.
