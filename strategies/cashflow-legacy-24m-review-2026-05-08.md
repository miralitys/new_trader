# Cashflow Legacy 24M Review

Дата: 2026-05-08

Период: `2024-05` - `2026-04`, последние 24 полных месяца.

Цель: посмотреть те же старые cashflow-ветки не на 36 месяцах, а на 24 месяцах:

- `ONE / SPELL 10%`;
- `MOVR / CHZ / MANA 4%`;
- варианты с добавлением `RIF`.

`SIFUSDT` в локальном futures inventory не найден. В проекте сильным кандидатом был `RIFUSDT`, поэтому ниже используется RIF.

## Короткий вывод

На 24 месяцах картина мягче, чем на 36 месяцах.

`ONE / SPELL 10%` на базе действительно выглядит рабочей recent-window стратегией: `24/24` месяцев закрывают цель `$100`. Но stress исполнения показывает, что запас тонкий: при maker `0.03%` остается только `17/24`, а при maker `0.03% + slippage 0.005%` только `15/24`.

Добавление RIF в `ONE / SPELL` улучшает fee-only stress: `23/24` вместо `17/24`, но не спасает slippage-stress и повышает просадку.

`MOVR / CHZ / MANA 4%` на 24 месяцах не проходит как cashflow: старая версия дает только `6/24` месяцев с `$40+`.

Лучший 24M вариант вокруг RIF для цели `$40` - `CHZ 20% / RIF 80%`, scale `6`, loss stop `5%`. Он дает `14/24`, то есть лучше старого MOVR-портфеля, но это все равно не стабильная ежемесячная касса.

### Вывод по ONE / SPELL

Если смотреть только 24 месяца, лучший вариант для исследования:

`ONE 10% / RIF 10% / SPELL 80%`, scale `7`, monthly loss stop `35%`.

Но это не спокойная стратегия. Это high-risk recent-window режим. Его нельзя ставить в один уровень с основными cashflow-стратегиями без отдельного paper/live execution теста.

### Вывод по MOVR / CHZ / MANA

Старый MOVR-портфель на 24 месяцах не подходит для ежемесячной кассы.

RIF-версия лучше, но тоже не решает задачу: `14/24` месяцев - это нормально для watchlist, но не для cashflow, где нам нужен почти каждый месяц.

## Итоговый статус

| Направление | 24M статус | Что делать |
|---|---|---|
| ONE / SPELL 10% | Рабочий recent-window high-risk | Оставить в архиве/watchlist, не делать основной |
| ONE / RIF / SPELL 10% | Лучше старого по fee-only stress | Можно отдельно наблюдать, но slippage ломает |
| MOVR / CHZ / MANA 4% | Не проходит 24M | Не брать как cashflow |
| CHZ / RIF 4% | Лучший RIF-кандидат на 24M, но только `14/24` | Watchlist, не cashflow |
