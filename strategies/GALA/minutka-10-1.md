# Минутка 10.1

## Название

GALAUSDT 1m LONG 10 with TP1 + breakeven + trailing stop

## Статус

Зафиксирована как `Минутка 10.1`.

Это trailing-версия `Минутки 10`, то есть лучшего найденного отдельного
LONG-кандидата.

## Идея

Минутка 10 уже проходит годовую проверку, но ее доходность слабее SHORT-ветки.
Здесь проверяется, поможет ли динамический выход вытянуть больше движения из
удачных LONG-сделок.

Логика:

- старый TP `0.25%` становится TP1-триггером;
- на TP1 закрывается `50%` позиции;
- stop по остатку переносится в breakeven с буфером;
- остаток ведется trailing stop `0.15%`;
- старый initial SL и time-stop остаются защитой.

Для LONG trailing считается так:

```text
trailing_stop = highest_price_since_entry * (1 - trailing_pct)
```

## Основные параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `LONG` |
| База | `Минутка 10` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `18%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Фильтр сигнала | `long_score >= 50 + smart_long_filter` |
| Initial SL | `4.0%` |
| TP1 trigger | `0.25%` |
| TP1 partial close | `50%` позиции |
| Breakeven buffer | `0.04%` |
| Trailing stop | `0.15%` |
| Time-stop | `90 минут` |
| Дневной автостоп | `2%` |

## Фильтр рыночного режима

| Фильтр | Значение |
|---|---:|
| ATR% | `>= 0.25%` |
| Дистанция close от EMA200 | от `-1.5%` до `+1.5%` |
| Доходность за 7 дней | от `-40%` до `+10%` |

```bash
python3 scripts/trailing_stop_strategies.py \

  --save-summary data/trailing_stop_summary.csv
```

## Вывод

Минутка 10.1 остается плюсовой на всех окнах, но как отдельный LONG она слабее
базовой Минутки 10 по доходности.

Главный плюс 10.1 - ниже годовая просадка:

```text
10:   +14.07%, Max DD 18.91%
10.1:  +8.49%, Max DD 11.44%
```

Практический статус: defensive LONG-версия. Она полезна, если важнее снизить
просадку, но не является улучшением по доходности.
