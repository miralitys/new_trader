# Минутка 7.4

## Название

GALAUSDT 1m SHORT 7.3 with TP1 + breakeven + trailing stop

## Статус

Зафиксирована как `Минутка 7.4`.

Это версия `Минутки 7.3`, где фиксированный тейк заменен на динамический
выход:

```text
TP1 -> частичная фиксация -> breakeven stop -> trailing stop по остатку.
```

## Идея

Базовая 7.3 хорошо зарабатывает, но на годе упирается почти ровно в лимит
просадки `20%`. Поэтому здесь проверяется не увеличение агрессии, а улучшение
качества выхода.

Старый тейк `0.28%` теперь не закрывает всю сделку. Он становится триггером:

- при достижении TP1 закрывается `50%` позиции;
- стоп по остатку переносится в безубыток с буфером под комиссии;
- остаток позиции ведется trailing stop;
- если цена продолжает падать, прибыль растет;
- если цена откатывает, остаток закрывается по trailing stop.

Для SHORT trailing считается зеркально:

```text
trailing_stop = lowest_price_since_entry * (1 + trailing_pct)
```

## Основные параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| База | `Минутка 7.3` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `24%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Фильтр сигнала | `short_score >= 40` |
| Initial SL | `4.0%` |
| TP1 trigger | `0.28%` |
| TP1 partial close | `50%` позиции |
| Breakeven buffer | `0.04%` |
| Trailing stop | `0.15%` |
| Time-stop | `120 минут` |
| Дневной автостоп | `2%` |

## Фильтр рыночного режима

Сделка разрешена только если на закрытой сигнальной свече выполнены все условия:

| Фильтр | Значение |
|---|---:|
| ATR% | `>= 0.25%` |
| Дистанция close от EMA200 | от `-1.5%` до `+1.5%` |
| Доходность за 7 дней | от `-40%` до `+10%` |

## Вывод

Минутка 7.4 - самая интересная trailing-версия.

На 365 днях доходность почти не изменилась относительно 7.3:

```text
7.3: +52.53%, Max DD 19.99%
7.4: +52.37%, Max DD 11.08%
```

То есть trailing не дал больше прибыли, зато резко улучшил форму риска.
Практический статус: 7.4 стоит считать защитной рабочей версией 7.3.
