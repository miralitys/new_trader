# Минутка 4

## Название

GALAUSDT 1m LONG defensive maker scalp

## Идея

Минутка 4 нужна как LONG-баланс к SHORT-минуткам. Цель: не максимальная
доходность, а стабильный плюсовой LONG-вариант с умеренной просадкой.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `LONG` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `3%` |
| Выход по времени | `90 минут` |
| Фильтр сигнала | `long_score >= 50` |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 90 --direction long \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --long-threshold 50 \
  --long-tp-pct 0.0025 \
  --long-sl-pct 0.030 \
  --time-stop-min 90
```

| Return | Win rate | PF | Max DD |

| +30.01% | 87.21% | 1.18 | 9.30% |

## Вывод

Минутка 4 является спокойной LONG-версией. Она слабее SHORT-вариантов по
доходности, но дает отдельную LONG-механику с умеренной просадкой.

Длинная проверка ухудшает оценку: на 180 днях минутка еще плюсовая, но
просадка вырастает до `24.19%`; на 365 днях минутка становится минусовой.
Поэтому минутка 4 не проходит годовую проверку устойчивости.
