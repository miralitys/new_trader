# Минутка 2.1

## Название

GALAUSDT 1m SHORT defensive maker scalp + daily loss kill switch

## Идея

Минутка 2.1 является модификацией минутки 2. Вся торговая логика остается
прежней, но добавляется дневной автостоп:

- если реализованный убыток за UTC-день превысил `2%` от equity на начало дня,
  новые сделки до конца дня больше не открываются;
- уже открытая сделка не закрывается принудительно этим правилом;
- автостоп срабатывает после закрытия сделки.

Поскольку размер позиции в минутке 2 равен `25%`, дневной автостоп срабатывает
редко.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `2%` |
| Выход по времени | `90 минут` |
| Фильтр сигнала | `short_score >= 55` |
| Дневной автостоп | `2%` убытка за день |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --short-threshold 55 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.020 \
  --time-stop-min 90 \
  --daily-loss-stop-pct 0.02
```

## Вывод

Дневной автостоп почти не меняет минутку 2, потому что позиция уже снижена до
`25%`, и дневной убыток `2%` достигается редко. На 365 днях минутка 2.1
немного лучше минутки 2, но годовую устойчивость все равно не проходит.
