# Минутка G5.1

## Название

GALAUSDT 1m LONG high win rate + daily loss kill switch

## Идея

Минутка G5.1 является модификацией минутки G5. Торговая логика остается
прежней, но добавляется дневной автостоп:

- если реализованный убыток за UTC-день превысил `2%` от equity на начало дня,
  новые сделки до конца дня больше не открываются;
- открытая сделка не закрывается принудительно;
- автостоп срабатывает после закрытия сделки.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `LONG` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `4%` |
| Выход по времени | `240 минут` |
| Фильтр сигнала | `long_score >= 50` |
| Дневной автостоп | `2%` убытка за день |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --direction long \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --long-threshold 50 \
  --long-tp-pct 0.0025 \
  --long-sl-pct 0.040 \
  --time-stop-min 240 \
  --daily-loss-stop-pct 0.02
```

## Вывод

Дневной автостоп почти не улучшает G5. Он немного снижает годовую просадку и
убыток, но ухудшает часть коротких периодов. Минутка G5.1 не проходит
годовую проверку устойчивости.
