# Минутка 6.1

## Название

GALAUSDT 1m combined S3 + G5 with daily kill stop

## Идея

Минутка 6.1 - это минутка 6 с дневным автостопом:
если за один UTC-день реализованный убыток превысил `2%` от баланса на начало
дня, новые входы до конца этого дня запрещаются.

Важно: дневной автостоп не закрывает уже открытую позицию принудительно. Он
только останавливает новые сделки после плохого дневного результата.

## Общие параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `both` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Одновременных позиций | `1` |
| Дневной автостоп | `2%` |

## LONG-часть

| Параметр | Значение |
|---|---:|
| Источник | Минутка G5 |
| Фильтр сигнала | `long_score >= 50` |
| Тейк | `0.25%` |
| Стоп | `4%` |
| Выход по времени | `240 минут` |

## SHORT-часть

| Параметр | Значение |
|---|---:|
| Источник | Минутка 3 |
| Фильтр сигнала | `short_score >= 40` |
| Тейк | `0.25%` |
| Стоп | `3%` |
| Выход по времени | `180 минут` |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --direction both \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --long-threshold 50 \
  --long-tp-pct 0.0025 \
  --long-sl-pct 0.040 \
  --long-time-stop-min 240 \
  --short-threshold 40 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.030 \
  --short-time-stop-min 180 \
  --daily-loss-stop-pct 0.02
```

## Вывод

Дневной автостоп немного уменьшил количество сделок и чуть улучшил 180 дней
относительно минутки 6, но проблему не решил. На 365 днях минутка остается
сильно минусовой: `PF < 1`, отрицательное ожидание на сделку и просадка около
`65%`.

Минутка 6.1 сохранена как проверенный вариант, но не как готовая рабочая
минутка. Главная проблема прежняя: без фильтра рыночного режима объединение
LONG и SHORT продолжает торговать в периоды, где микроскальп теряет edge.
