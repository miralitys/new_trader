# Минутка 11.3

## Название

GALAUSDT 1m portfolio 11.2 with trailing exits

## Статус

Зафиксирована как `Минутка 11.3`.

Это портфельная trailing-версия `Минутки 11.2`.

## Идея

В 11.2 портфель состоит из двух рабочих модулей:

- `Минутка 10` LONG;
- `Минутка 7.3` SHORT с усилением `x1.5`.

В 11.3 эти модули заменены на их trailing-версии:

- `Минутка 10.1` LONG trailing;
- `Минутка 7.4` SHORT trailing с усилением `x1.5`.

Портфельное правило остается тем же:

```text
Одновременно может быть открыта только одна GALA-сделка.
```

## Состав

| Модуль | Пара | Таймфрейм | Направление | Роль |
|---|---|---:|---|---|
| GALA LONG Minutka 10.1 | `GALAUSDT` | `1m` | LONG | Защитный LONG trailing-модуль |
| GALA SHORT Minutka 7.4 x1.5 | `GALAUSDT` | `1m` | SHORT | Основной SHORT trailing-модуль |

## Портфельный слой

| Параметр | Значение |
|---|---:|
| Начальный баланс | `$1000` |
| Модель портфеля | `compound` |
| Portfolio scale | `0.9` |
| Max open | `1` |
| Вес LONG 10.1 | `1.0` |
| Вес SHORT 7.4 | `1.5` |

Эффективное влияние модулей:

```text
LONG 10.1: 18% * 0.9 = 16.2%
SHORT 7.4: 24% * 0.9 * 1.5 = 32.4%
```

## Dynamic exit

| Параметр | LONG 10.1 | SHORT 7.4 |
|---|---:|---:|
| TP1 trigger | `0.25%` | `0.28%` |
| TP1 partial close | `50%` | `50%` |
| Breakeven buffer | `0.04%` | `0.04%` |
| Trailing stop | `0.15%` | `0.15%` |
| Initial SL | `4.0%` | `4.0%` |
| Time-stop | `90m` | `120m` |

## Вывод

Минутка 11.3 - не более доходная версия 11.2, а более защитная.

Годовой результат:

```text
11.2 compound: +68.11%, Max DD 16.43%
11.3 trailing: +46.56%, Max DD 6.96%
```

Если выбирать по максимальной доходности, сильнее остается 11.2. Если выбирать
по спокойствию кривой и просадке, 11.3 выглядит лучше.

Практический статус: защитный портфельный кандидат.
