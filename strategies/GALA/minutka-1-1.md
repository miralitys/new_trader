# Минутка 1.1

## Название

GALAUSDT 1m SHORT maker scalp + daily loss kill switch

## Идея

Минутка 1.1 является модификацией минутки 1. Вся торговая логика
остается прежней, но добавляется дневной автостоп:

- если реализованный убыток за UTC-день превысил `2%` от equity на начало дня,
  новые сделки до конца этого дня больше не открываются;
- уже открытая сделка не закрывается принудительно этим правилом;
- автостоп срабатывает после закрытия сделки, когда дневной убыток уже
  зафиксирован.

Цель: обрезать плохие дни и уменьшить разрушительные серии стопов.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `100%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `2%` |
| Выход по времени | `90 минут` |
| Фильтр сигнала | `short_score >= 55` |
| Дневной автостоп | `2%` убытка за день |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 1.0 \
  --short-threshold 55 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.020 \
  --time-stop-min 90 \
  --daily-loss-stop-pct 0.02
```

## Вывод

Дневной автостоп заметно улучшает минутку 1:

- уменьшает количество сделок;
- уменьшает просадку;
- улучшает 180-дневный результат;
- снижает годовой убыток.

Но минутка 1.1 все равно не проходит годовую проверку устойчивости. На 365
днях результат остается отрицательным, а просадка слишком высокая. Это
улучшение минутки 1, но не финальная рабочая версия.
