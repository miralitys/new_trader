# Минутка 2

## Название

GALAUSDT 1m SHORT defensive maker scalp

## Идея

Это защитная версия минутки 1. Сигнал, тейк, стоп и выход по времени остаются
такими же, но размер позиции снижен с `100%` до `25%` текущего баланса.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `2%` |
| Выход по времени | `90 минут` |
| Фильтр сигнала | `short_score >= 55` |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 90 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --short-threshold 55 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.020 \
  --time-stop-min 90
```

## Результаты

| Период | Сделок | Return | Win rate | PF | Max DD |
|---|---:|---:|---:|---:|---:|
| 90d | 4 539 | +39.41% | 87.05% | 1.21 | 12.30% |
| 180d | 8 380 | +16.60% | 85.47% | 1.05 | 27.01% |
| 365d | 15 234 | -45.87% | 83.89% | 0.89 | 66.52% |

## Сравнение с минуткой 1

| Вариант | Размер позиции | Return 90d | Max DD 90d |
|---|---:|---:|---:|
| Минутка 1 | 100% | +259.82% | 41.58% |
| Минутка 2 | 25% | +39.41% | 12.30% |

## Вывод

Минутка 2 снижает риск за счет меньшего размера позиции. Доходность ниже,
но просадка становится намного спокойнее.

Длинная проверка показывает, что снижение позиции помогает, но не решает
проблему полностью. На 180 днях минутка еще плюсовая, но просадка уже
`27.01%`; на 365 днях минутка становится минусовой. Поэтому минутка 2
тоже не проходит годовую проверку устойчивости.
