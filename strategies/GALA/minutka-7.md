# Минутка 7

## Название

GALAUSDT 1m SHORT regime-filtered conservative

## Идея

Минутка 7 - это консервативная SHORT-минутка с фильтром рыночного режима.

Цель была не максимальная доходность, а выполнение трех условий на всех
проверочных окнах:

- доходность выше нуля;
- win rate выше `85%`;
- максимальная просадка не выше `13%`.

## Основные параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `10%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Фильтр сигнала | `short_score >= 40` |
| Тейк | `0.25%` |
| Стоп | `3%` |
| Выход по времени | `180 минут` |
| Дневной автостоп | `2%` |

## Фильтр рыночного режима

Сделка разрешена только если на закрытой сигнальной свече выполнены все условия:

| Фильтр | Значение |
|---|---:|
| ATR% | `>= 0.25%` |
| Дистанция close от EMA200 | от `-2.5%` до `+2.5%` |
| Доходность за 1 день | от `-20%` до `+5%` |
| Доходность за 7 дней | от `-35%` до `+5%` |

Смысл фильтра: не шортить любой сигнал подряд, а брать только режим, где цена
достаточно волатильна, но не находится слишком далеко от средней и не идет в
сильный недельный памп.

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --warmup-days 7 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.10 \
  --short-threshold 40 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.030 \
  --short-time-stop-min 180 \
  --daily-loss-stop-pct 0.02 \
  --filter-atr-min-pct 0.0025 \
  --filter-dist-ema200-min -0.025 \
  --filter-dist-ema200-max 0.025 \
  --filter-return-1d-min -0.20 \
  --filter-return-1d-max 0.05 \
  --filter-return-7d-min -0.35 \
  --filter-return-7d-max 0.05
```

`--warmup-days 7` нужен, чтобы недельный фильтр считался на предыдущей истории,
а не обнулял первые свечи внутри тестового окна.

## Вывод

Минутка 7 проходит заданные условия на всех проверочных окнах: 7, 30, 90,
180 и 365 дней. Просадка на годе `10.96%`, win rate `90.12%`, доходность
остается положительной.

Но запас прочности маленький: на 365 днях `PF = 1.02`, а ожидание на сделку
всего `+0.0004%`. Это значит, что минутка выглядит лучше предыдущих вариантов
по контролю просадки, но очень чувствительна к ухудшению исполнения, комиссии и
реальному неполному maker-заполнению.

Минутка 7 сохранена как лучший на текущий момент консервативный кандидат, но
следующий обязательный шаг - стресс-тест с худшей комиссией и проскальзыванием.
