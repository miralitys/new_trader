# Минутка 1

## Название

GALAUSDT 1m SHORT maker scalp

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `100%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `2%` |
| Выход по времени | `90 минут` |
| Фильтр сигнала | `short_score >= 55` |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 90 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 1.0 \
  --short-threshold 55 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.020 \
  --time-stop-min 90
```

## Вывод

Минутка агрессивная. Доходность высокая, но просадка большая из-за позиции
на весь баланс и широкого стопа относительно тейка. Подходит как базовая
агрессивная SHORT-модель, но не как спокойный риск-профиль.

Длинная проверка ухудшает оценку минутки: на 180 днях доходность еще
плюсовая, но просадка достигает `73.34%`; на 365 днях минутка становится
глубоко минусовой. Поэтому минутка 1 не проходит проверку устойчивости на
годовом горизонте.
