# Минутка 4.1

## Название

GALAUSDT 1m LONG defensive maker scalp + daily loss kill switch

## Идея

Минутка 4.1 является модификацией минутки 4. Торговая логика остается
прежней, но добавляется дневной автостоп:

- если реализованный убыток за UTC-день превысил `2%` от equity на начало дня,
  новые сделки до конца дня больше не открываются;
- открытая сделка не закрывается принудительно;
- автостоп срабатывает после закрытия сделки.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `LONG` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `3%` |
| Выход по времени | `90 минут` |
| Фильтр сигнала | `long_score >= 50` |
| Дневной автостоп | `2%` убытка за день |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --direction long \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --long-threshold 50 \
  --long-tp-pct 0.0025 \
  --long-sl-pct 0.030 \
  --time-stop-min 90 \
  --daily-loss-stop-pct 0.02
```

## Вывод

Дневной автостоп почти не меняет минутку 4. На 180 и 365 днях он слегка
улучшает результат и просадку, но эффект небольшой. Минутка 4.1 все равно не
проходит годовую проверку устойчивости.
