# Минутка G5

## Название

GALAUSDT 1m LONG high win rate

## Идея

G5 является развитием минутки 4. Цель: увеличить win rate LONG-версии, даже
если доходность станет ниже.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `LONG` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `4%` |
| Выход по времени | `240 минут` |
| Фильтр сигнала | `long_score >= 50` |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 90 --direction long \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --long-threshold 50 \
  --long-tp-pct 0.0025 \
  --long-sl-pct 0.040 \
  --time-stop-min 240
```

## Результаты

| Период | Сделок | Return | Win rate | PF | Max DD |
|---|---:|---:|---:|---:|---:|
| 7d | 232 | +0.67% | 90.95% | 1.06 | 2.31% |
| 30d | 1 206 | +29.25% | 93.86% | 1.76 | 2.31% |
| 90d | 3 166 | +37.51% | 92.04% | 1.30 | 7.83% |
| 180d | 5 656 | +19.17% | 90.93% | 1.08 | 20.32% |
| 365d | 10 465 | -42.80% | 89.78% | 0.87 | 62.45% |

## Stress test

`slippage=0.01%` на 90 дней:

| Return | Win rate | PF | Max DD |
|---:|---:|---:|---:|
| +27.04% | 92.04% | 1.22 | 8.81% |

## Сравнение с минуткой 4

| Минутка | Return 90d | Win rate 90d | Max DD 90d |
|---|---:|---:|---:|
| Минутка 4 | +45.11% | 87.21% | 7.69% |
| Минутка G5 | +37.51% | 92.04% | 7.83% |

## Вывод

G5 повышает win rate LONG-версии, но снижает доходность относительно минутки
4. На 7 днях она почти в ноль, зато на 30 и 90 днях выглядит устойчиво.

Длинная проверка показывает, что G5 тоже не проходит годовую устойчивость. На
180 днях минутка еще плюсовая, но просадка уже `20.32%`; на 365 днях итог
становится отрицательным.
