# Минутка 3.1

## Название

GALAUSDT 1m SHORT high win rate + daily loss kill switch

## Идея

Минутка 3.1 является модификацией минутки 3. Торговая логика остается
прежней, но добавляется дневной автостоп:

- если реализованный убыток за UTC-день превысил `2%` от equity на начало дня,
  новые сделки до конца дня больше не открываются;
- открытая сделка не закрывается принудительно;
- автостоп срабатывает после закрытия сделки.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `3%` |
| Выход по времени | `180 минут` |
| Фильтр сигнала | `short_score >= 40` |
| Дневной автостоп | `2%` убытка за день |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 365 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --short-threshold 40 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.030 \
  --time-stop-min 180 \
  --daily-loss-stop-pct 0.02
```

## Вывод

Дневной автостоп улучшает минутку 3: немного снижает просадку и улучшает
результаты на 90, 180 и 365 днях. Но годовая проверка все равно остается
отрицательной, поэтому минутка 3.1 не является финально устойчивой.
