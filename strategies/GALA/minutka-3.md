# Минутка 3

## Название

GALAUSDT 1m SHORT high win rate

## Идея

Цель минутки 3: повысить win rate относительно минутки 2, оставить
доходность плюсовой и удержать просадку ниже `13%`.

## Параметры

| Параметр | Значение |
|---|---:|
| Пара | `GALAUSDT` |
| Таймфрейм | `1m` |
| Направление | `SHORT` |
| Вход | Лимитный у текущей цены |
| Размер позиции | `25%` текущего баланса |
| Комиссия | `0.02%` вход + `0.02%` выход |
| Проскальзывание | `0` |
| Тейк | `0.25%` |
| Стоп | `3%` |
| Выход по времени | `180 минут` |
| Фильтр сигнала | `short_score >= 40` |

## Команда

```bash
python3 scripts/gala_mb_backtest.py --market futures_archive --days 90 --direction short \
  --entry-mode maker_limit \
  --limit-entry-offset-pct 0 \
  --limit-entry-timeout-min 1 \
  --fee-pct 0.0002 \
  --slippage-pct 0 \
  --position-pct 0.25 \
  --short-threshold 40 \
  --short-tp-pct 0.0025 \
  --short-sl-pct 0.030 \
  --time-stop-min 180
```

## Результаты

|---|---:|---:|---:|---:|---:|
| 7d | 333 | +7.68% | 92.19% | 1.84 | 1.90% |
| 30d | 1 435 | +25.42% | 91.64% | 1.49 | 2.64% |
| 90d | 4 149 | +61.78% | 91.30% | 1.34 | 5.72% |
| 180d | 7 591 | +33.43% | 89.91% | 1.11 | 22.95% |
| 365d | 13 853 | -31.59% | 88.82% | 0.92 | 61.07% |

## Stress test

`slippage=0.01%` на 90 дней:

| Return | Win rate | PF | Max DD |
|---:|---:|---:|---:|
| +45.85% | 91.30% | 1.27 | 6.44% |

## Вывод

Минутка 3 выглядит наиболее спокойной среди SHORT-вариантов: высокий win
rate, плюсовая доходность, низкая просадка. Плата за это: более широкий стоп
и более долгий выход по времени.

Длинная проверка меняет оценку: минутка остается плюсовой на 180 днях, но
просадка растет до `22.95%`; на 365 днях минутка становится минусовой. Это
лучше, чем минутка 1 и 2 по глубине ущерба, но годовую устойчивость минутка
3 тоже не проходит.
